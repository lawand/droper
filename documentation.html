General Usage Tips

When downloading files, you can close the download dialog and the download won't be interrupted, and you can check the download progress later by selecting the Show Active Download option.

You can block Droper's access to your account, in case you forget to remove the authentication when using Droper on a friend's device, by logging into your Dropbox account in dropbox.com and selecting the My Apps option in the account's settings and removing Droper's access. You can give the access back by logging in again using Droper.

Uninstalling Droper doesn't remove the authentication, you must do so manually, using the "Forget Authentication" option, for more info check out the tutorial.

There seem to be some issues when dealing with some files and folders which names contain symbols.

The current version isn't optimized for minimal network traffic, use only with WLAN networks when using on a mobile device.
