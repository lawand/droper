Droper - Symbian Dropbox Client 0.3.1

Introduction

Droper is a Qt4 GPLv3 open source cross-platform application which helps Dropbox users access and modify their synced files and folders.

It is designed for mobile phones (running the Symbian platform), but is also ported for desktop platforms like Windows, in which case it can be used as a simple, light-weight Dropbox client.

(Latest version: 0.3.1, released on 2011-04-14)

Changelog

Changes from 0.2.1:

  • Added an option to upload files.
  • Lots of GUI enhancements including an experimental kinetic scrolling feature for Symbian.
  • An option to view file properties which include the date of last modification.
  • Lots of bug fixes and other minor enhancements.
