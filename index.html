Introduction

Droper is a Qt4 GPLv3 open source cross-platform application which helps Dropbox users access and modify their synced files and folders.

Download

Windows

Download Droper v0.1 for Windows - 4.6MB

  • OpenSSL (version 0.9.7 or later) should be installed first.

Source Code

You can download the source code of v0.1 in either zip or tar formats.

Checkout the git repository on Github: lawand/droper.

Usage Notes

The current version of Droper doesn't allow downloading large files, and the progress bar isn't visible yet.

Also, there seem to be some issues when dealing with some files and folders which names contain symbols.

The current version isn't optimized for minimal network traffic, use only with WLAN networks when using on a mobile device.

Notes For Developers

Droper is a C++/Qt4 application, and is tested with Qt 4.7.0. Droper also requires OpenSSL (which is also required by Qt)

If you try to compile Droper, you'll notice that the consumerdata.cpp file is missing, it is a simple file that contains the values of the consumer key and secret. This file wasn't uploaded to the repository for security reasons. For more info search for and learn about OAuth.
