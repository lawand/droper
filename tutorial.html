Droper - Symbian Dropbox Client

Introduction

This is a step-by-step tutorial that describes the usage of Droper's main functions, based on v0.2.1 for Symbian.

Starting Droper

Tap Droper's icon found in the menu, then enter your Dropbox account's email and password.

Next, you'll see droper's main window.

Navigation

Double tap the item you want to enter.

Then you'll see that folder's contents.

And to return to the parent folder, double tap the "Up" item.

That's when you'll access the parent folder.

Cut, Copy And Paste

To copy a file or folder, tap it and hold until you see the menu, then choose Copy.

Then navigate into the folder you want to paste in, and tap the current folder button (found next to the current folder's name) then choose Paste.

Then the file or folder would get copied, and you see it among the current folder's items.

Cutting is done in a similar fashion.

Folder Creation

To create a new folder, navigate into the folder in which you want to create the new folder, then tap the current folder button, and choose Create Folder.

Enter the new folder's name.

Then tap OK, and the folder will be created.

Refreshing The Current Folder

To reload the current folder's items, tap the current folder button, and choose Refresh.

Renaming

To change the name of a file or folder, tap it and hold until you see the menu, then choose Rename.

Enter the new name and tap OK.

The item will be renamed.

Deletion

To remove a file or folder, tap it and hold until you see the menu, then choose Delete.

Confirm by tapping Yes.

The item will be deleted.

Downloading

To download a file, tap it and hold until you see the menu, then choose Download.

You'll see a download dialog, tap browse to choose the location into which you want to save the file.

Choose the location.

Then tap start.

After waiting for the download to finish, you'll see a message box saying the the download is done.

View Account Information

To view available space and other account information, tap the options menu, and choose Options.

Then choose Show Account Info.

A message box containing all account info will appear momentarily.

Authentication Removal

To remove the stored authentication (effectively making the application forget your email and password), tap the options menu, and choose Options.

Then choose Forget Authentication.

Then close the program.
