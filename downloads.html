Droper - Symbian Dropbox Client 0.3.1

Symbian

Download Droper v0.3.1 for Symbian - 187KB

  • Compatible with all Nokia Symbian smartphones running Symbian^3 or S60v5 (in other words, all Nokia Symbian smartphones equipped with a touch screen). For more info about your device, check out Forum Nokia.
  • Requires installing Qt 4.6.3 first for S60v5 devices (but not for Symbian^3 devices). You can get Qt 4.6.3 here.
  • This version will soon be available at Ovi Store.

Get Droper v0.2.1 for Symbian at Ovi Store here.

  • If you have manually installed a 4.7.x or later version of Qt, it's recommended for you to uninstall it first.

These two version are not compatible. You can't install them simultaneously, and you should uninstall one first if you want to install the other.

Windows

Download Droper v0.2.1 for Windows - 4.7MB

  • OpenSSL (version 0.9.7 or later) should be installed first. (Using default options during install is required)

Version 0.3.1 for Windows will be available shortly.
